Project

Gem's Fictoon

Gem's Fictoon is a mobile-web entertainment application featuring a guiding original character (Gem) where users are free to create their own work of fiction. As for readers, different reading comfortability is offered such as background music and visual customization. As for authors, a writing guide will be provided including reminders and a grammar checker. The application is managed by the admin and monitored for safe access.
